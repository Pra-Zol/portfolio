# Prajwal Khanal — Personal Portfolio

A full-stack personal portfolio website built with React, Node.js, Express, and PostgreSQL.

🌐 **Live Demo:** [prajwalkhanal.vercel.app](https://prajwalkhanal.vercel.app)

---

## About

Hi, I'm Prajwal Khanal — a Full-Stack Developer and Bachelor of IT student at Kent Institute Australia, based in Sydney. This portfolio showcases my projects, skills, and experience in software development and aged care.

---

## Built With

- **Frontend:** React, CSS3, HTML5
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (Neon)
- **Deployment:** Vercel (frontend), Railway (backend)

---

## Features

- Responsive design — works on mobile and desktop
- Dark and light mode toggle
- Animated hero section with typing effect
- Filterable projects section
- Working contact form (saves to PostgreSQL database)
- Email notifications on form submission
- Downloadable CV button

## Run Locally

**Clone the repo:**
```bash
git clone https://github.com/Pra-Zol/portfolio.git
cd portfolio
```

**Start the backend:**
```bash
cd backend
npm install
cp .env.example .env
# Add your DATABASE_URL in .env
npm run dev
```

**Start the frontend:**
```bash
cd frontend
npm install
npm start
```
